## Étape 1 — Enrichir l'environnement `SimpleTradingEnv`

### Problème
L'observation actuelle = 5 diffs de prix bruts. L'agent ne voit ni tendance, ni volatilité, ni momentum.

### Solution
Passer l'observation de **5 → 10 features** en utilisant les données déjà calculées par `features.py` dans le DataFrame en entrée :

| Feature | Description | Source |
|---------|-------------|--------|
| `price_diff_1..5` | 5 derniers diffs de prix (existant) | `Close` |
| `returns_1..5` | 5 derniers rendements % | `Close` / shift |

**Observation space** : `Box(shape=(10,))` au lieu de `(5,)`

**Reward enrichie** :
- Ajouter une pénalité pour les actions HOLD successives (éviter l'inaction)
- Normaliser la reward par la volatilité récente (ATR-like) pour que l'agent ne soit pas biaisé par l'échelle de prix

### Détails d'implémentation dans `tensortrade_model.py`

```python
class SimpleTradingEnv(gym.Env):
    def __init__(self, prices):
        super().__init__()
        self.prices = prices
        self.current_step = 0
        self.hold_count = 0
        self.action_space = gym.spaces.Discrete(3)
        self.observation_space = gym.spaces.Box(
            low=-np.inf, high=np.inf, shape=(10,), dtype=np.float32
        )

    def step(self, action):
        self.current_step += 1
        done = self.current_step >= len(self.prices) - 1

        reward = 0
        if not done:
            price_change = self.prices[self.current_step] - self.prices[self.current_step - 1]
            # Normalize by recent volatility (ATR-like)
            window = self.prices[max(0, self.current_step - 14):self.current_step + 1]
            atr = np.mean(np.abs(np.diff(window))) if len(window) > 1 else 1.0
            atr = max(atr, 1e-6)  # avoid division by zero

            if action == 1:  # Buy
                reward = price_change / atr
                self.hold_count = 0
            elif action == 2:  # Sell
                reward = -price_change / atr
                self.hold_count = 0
            else:  # Hold
                self.hold_count += 1
                # Small penalty for extended holds to encourage decisiveness
                reward = -0.01 * min(self.hold_count, 10)

        obs = self._get_obs()
        return obs, reward, done, False, {}

    def reset(self, seed=None, options=None):
        self.current_step = 10  # Need 10 bars for returns window
        self.hold_count = 0
        return self._get_obs(), {}

    def _get_obs(self):
        idx = self.current_step
        # 5 price diffs
        diffs = np.diff(self.prices[idx - 5:idx + 1])
        # 5 returns (% change)
        recent = self.prices[idx - 5:idx + 1]
        returns = np.diff(recent) / recent[:-1]
        return np.concatenate([diffs, returns]).astype(np.float32)
```

**Note** : On garde l'env dans le fichier (class interne) comme actuellement. L'extraire dans un module séparé est un nice-to-have hors scope.

## Étape 2 — Persistance du modèle PPO (save/load)

### Problème
Le modèle est recréé à chaque appel. 193k timesteps d'apprentissage gaspillés.

### Solution
Sauvegarder/charger le modèle PPO via l'API native `stable-baselines3` (`model.save()` / `PPO.load()`).

### Design
- **Répertoire** : `data_cache/tensortrade/` (déjà couvert par `.gitignore` ligne 45)
- **Fichier** : `ppo_model.zip` (format natif SB3)
- **Stratégie** :
  - Si un modèle sauvegardé existe → le charger et fine-tuner avec `total_timesteps=500`
  - Si pas de modèle → créer un nouveau PPO et entraîner `total_timesteps=2000` (premier run plus long)
  - Après entraînement → sauvegarder systématiquement (écrase le précédent)
- **Horodatage** : Un fichier `metadata.json` à côté pour tracker la date de dernier entraînement, le nombre total de timesteps cumulés, et la version de l'environnement (observation shape)

### Implémentation dans `tensortrade_model.py`

```python
from pathlib import Path

_MODEL_DIR = Path("data_cache") / "tensortrade"
_MODEL_PATH = _MODEL_DIR / "ppo_model.zip"
_METADATA_PATH = _MODEL_DIR / "metadata.json"
_INITIAL_TIMESTEPS = 2000
_FINE_TUNE_TIMESTEPS = 500

def _load_metadata():
    if _METADATA_PATH.exists():
        with open(_METADATA_PATH, "r") as f:
            return json.load(f)
    return {"total_timesteps": 0, "last_trained": None, "obs_shape": None}

def _save_metadata(total_timesteps, obs_shape):
    _MODEL_DIR.mkdir(parents=True, exist_ok=True)
    metadata = {
        "total_timesteps": total_timesteps,
        "last_trained": datetime.now().isoformat(),
        "obs_shape": list(obs_shape),
    }
    with open(_METADATA_PATH, "w") as f:
        json.dump(metadata, f, indent=2)

def get_tensortrade_prediction(df):
    # ... validation ...
    prices = data[close_col].values
    env = SimpleTradingEnv(prices)

    _MODEL_DIR.mkdir(parents=True, exist_ok=True)

    if _MODEL_PATH.exists():
        metadata = _load_metadata()
        # Invalidate model if observation space changed
        if metadata.get("obs_shape") and metadata["obs_shape"] != list(env.observation_space.shape):
            logger.info("Observation space changed, retraining from scratch.")
            _MODEL_PATH.unlink(missing_ok=True)
        else:
            try:
                model = PPO.load(_MODEL_PATH, env=env)
                logger.info(f"Modèle PPO chargé depuis le cache ({metadata.get('total_timesteps', '?')} timesteps cumulés)")
                model.learn(total_timesteps=_FINE_TUNE_TIMESTEPS)
                metadata["total_timesteps"] = metadata.get("total_timesteps", 0) + _FINE_TUNE_TIMESTEPS
                _save_metadata(metadata["total_timesteps"], env.observation_space.shape)
                model.save(_MODEL_PATH)
                logger.info(f"Modèle PPO sauvegardé ({_FINE_TUNE_TIMESTEPS} timesteps ajoutés, total: {metadata['total_timesteps']})")
            except Exception as e:
                logger.warning(f"Erreur chargement modèle PPO, retraining from scratch: {e}")
                model = PPO("MlpPolicy", env, n_steps=64, batch_size=32, verbose=0)
                model.learn(total_timesteps=_INITIAL_TIMESTEPS)
                _save_metadata(_INITIAL_TIMESTEPS, env.observation_space.shape)
                model.save(_MODEL_PATH)

    else:
        logger.info("Aucun modèle PPO en cache, entraînement initial...")
        model = PPO("MlpPolicy", env, n_steps=128, batch_size=64, verbose=0)
        model.learn(total_timesteps=_INITIAL_TIMESTEPS)
        _save_metadata(_INITIAL_TIMESTEPS, env.observation_space.shape)
        model.save(_MODEL_PATH)
        logger.info(f"Modèle PPO initial sauvegardé ({_INITIAL_TIMESTEPS} timesteps)")

    # ... prédiction (inchangée) ...
```

### Paramètres PPO ajustés pour l'entraînement initial
- `n_steps=128` (au lieu de 64) → 16 updates au lieu de 8
- `batch_size=64` (au lieu de 32) → gradients plus stables
- `total_timesteps=2000` (au lieu de 500) pour le premier run

## Risques et mitigations

| Risque | Mitigation |
|--------|-----------|
| Chargement d'un modèle corrompu | try/except autour de `PPO.load()` avec fallback sur entraînement from scratch |
| Incompatibilité de shape (modèle ancien vs nouvel env 10 features) | Check `metadata.obs_shape` avant load, invalider si différent |
| Surcharge disque | Fichier unique `ppo_model.zip` écrasé à chaque fois (~1-5 MB), pas de croissance |
| Régression des performances | Les tests existants + bench valident le output format ; la confiance peut changer mais le format reste le même |
| Migration DB échoue | Mécanisme de recovery existant dans `_migrate_model_signals_table` (lignes 242-254) |
