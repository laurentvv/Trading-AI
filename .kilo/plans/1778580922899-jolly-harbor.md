## Phase 4 : Configurer HF_TOKEN sur Windows 11 PROD [A EXECUTER]

### Process etape par etape

#### Etape 1 : Generer la cle HF
1. Aller sur https://huggingface.co/settings/tokens
2. Se connecter / Creer un compte
3. Cliquer "New token"
4. Nom : `trading-ai-prod`
5. Type : **Read** (suffisant pour downloader les modeles)
6. Copier le token (format `hf_xxxxxxxxxxxxxxxxxxxxxx`)

#### Etape 2 : Ajouter en variable d'environnement Windows
```powershell

# Option B : Variable systeme (si service tourne en SYSTEM)
[System.Environment]::SetEnvironmentVariable("HF_TOKEN", "hf_xxxxxxxxxxxxxxxxxxxxxx", "Machine")
```

```powershell

```

Le service doit etre redemarre pour prendre en compte la nouvelle variable d'environnement.

#### Alternative : Ajouter dans `.env`
Si le projet utilise un fichier `.env` (python-dotenv), ajouter :
```

```
