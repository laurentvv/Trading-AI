## 🌟 প্রকল্প সম্পর্কে

এই প্রকল্পটি ETF ট্রেডিংয়ের জন্য একটি বিশেষজ্ঞ সিদ্ধান্ত-সমর্থন সিস্টেম, যা একটি ট্রাই-মোডাল হাইব্রিড এআই পদ্ধতি ব্যবহার করে। এটি একাধিক এআই দৃষ্টিভঙ্গি একত্রিত করে একটি বিস্তৃত এবং শক্তিশালী বিশ্লেষণ প্রদানের জন্য ডিজাইন করা হয়েছে।

### 🚀 ডুয়াল-টিকার কৌশল (বিশ্লেষণ বনাম ট্রেডিং)
সিস্টেমটি মডেলের নির্ভুলতা বাড়াতে একটি উদ্ভাবনী পদ্ধতি ব্যবহার করে:
- **হাই-ফিদেলিটি বিশ্লেষণ**: এআই মডেলগুলি **বৈশ্বিক রেফারেন্স সূচক** (নাসডাকের জন্য `^NDX`, WTI অপরিশোধিত তেলের জন্য `CL=F`) বিশ্লেষণ করে। এই সূচকগুলি দীর্ঘতর ইতিহাস এবং "বিশুদ্ধ" প্রবণতা অফার করে, যেখানে ট্রেডিং সময় বা ETF ফি সংক্রান্ত কোনো নয়েজ থাকে না।
- **ETF সম্পাদন**: বাস্তব অর্ডারগুলি **Trading 212**-এর সংশ্লিষ্ট টিকারে (`SXRV.DE`, `CRUDP.PA`) দেওয়া হয়, যেখানে পজিশন সাইজিংয়ের জন্য **T212 লাইভ প্রাইস** (পজিশন এপিআই-এর মাধ্যমে) ব্যবহার করা হয়।

### 🧠 হাইব্রিড এআই ইঞ্জিন
সিস্টেমটি আটটি ভিন্ন সংকেতকে একত্রিত করে:
1.  **ক্লাসিক কোয়ান্টিটেটিভ মডেল**: প্রযুক্তিগত এবং সামষ্টিক অর্থনৈতিক সূচকগুলির ওপর প্রশিক্ষিত RandomForest/GradientBoosting/LogisticRegression এনসেম্বল।
2.  **TimesFM 2.5 (Google Research)**: টাইম-সিরিজ পূর্বাভাসের জন্য অত্যাধুনিক ফাউন্ডেশন মডেল।
3.  **Oil-Bench মডেল (Gemma 4:e4b)**: জ্বালানি-বিশেষজ্ঞ মডেল যা **EIA** মৌলিক ডেটা (মজুদ, আমদানি, শোধনাগার ব্যবহার) এবং WTI ট্রেডিংয়ের সেন্টিমেন্টকে একত্রিত করে।
4.  **টেক্সচুয়াল LLM (Gemma 4:e4b)**: র ডেটার প্রাসঙ্গিক বিশ্লেষণ, **AlphaEar** স্কিলের মাধ্যমে রিয়েল-টাইম সংবাদ এবং গতিশীল **সামষ্টিক অর্থনৈতিক ওয়েব রিসার্চের** সংমিশ্রণ।
5.  **ভিজ্যুয়াল LLM (Gemma 4:e4b)**: প্রযুক্তিগত চার্ট (`enhanced_trading_chart.png`) সরাসরি বিশ্লেষণ।
6.  **সেন্টিমেন্ট বিশ্লেষণ**: আলফা ভ্যানটেজ এবং **AlphaEar** (Weibo, WallstreetCN) থেকে "হট" ট্রেন্ডের সমন্বয়ে হাইব্রিড বিশ্লেষণ।
7.  **বিকেন্দ্রীভূত ডেটা (Hyperliquid)**: *ফান্ডিং রেট* এবং *ওপেন ইন্টারেস্ট*-এর মাধ্যমে তেলের (WTI) ওপর ফটকামূলক সেন্টিমেন্ট বিশ্লেষণ।
8.  **ভিনসেন্ট গ্যানে মডেল**: সামষ্টিক অর্থনৈতিক তলা (bottoms) সনাক্ত করার জন্য ভূ-রাজনৈতিক এবং ক্রস-অ্যাসেট বিশ্লেষণ (WTI, Brent, Gas, DXY, MA200)।

লক্ষ্য হলো **নির্ভুলতা প্রথম** এই অগ্রাধিকারের ভিত্তিতে একটি চূড়ান্ত সিদ্ধান্ত (`BUY`, `SELL`, `HOLD`) তৈরি করা।

### 🧘 সিদ্ধান্ত দর্শন: "কগনিটিভ প্রুডেন্স"
সাধারণ ট্রেডিং অ্যালগরিদমগুলির মতো নয় যা অস্থিরতা বাড়লেই আতঙ্কিত হয়, এই সিস্টেমটি একজন সচেতন বিনিয়োগকারীর দৃষ্টিভঙ্গি প্রয়োগ করে:
- **শক্তিশালী ঐকমত্য প্রয়োজন**: একটি কোয়ান্টিটেটিভ মডেল (ক্লাসিক) হয়তো বিক্রির সংকেত (`SELL`) দিতে পারে, কিন্তু যদি কগনিটিভ মডেলগুলি (টেক্সট LLM, ভিশন, TimesFM) নিরপেক্ষ থাকে, তবে সিস্টেমটি `HOLD` করা পছন্দ করবে।
- **কনফিডেন্স ফিল্টার**: একটি মুভমেন্ট সিদ্ধান্ত (কেনা বা বিক্রি) কেবল তখনই বৈধ হয় যদি বৈশ্বিক আত্মবিশ্বাস একটি নিরাপত্তা থ্রেশহোল্ড (সাধারণত ৪০%) অতিক্রম করে। এর নিচে সিস্টেমটি সংকেতকে "নয়েজ" হিসেবে গণ্য করে এবং স্ট্যান্ডবাই থাকে।
- **মূলধন সুরক্ষা**: `VERY_HIGH` ঝুঁকি মোডে, `HOLD` একটি ঢাল হিসেবে কাজ করে। এটি একটি অস্থির বাজারে প্রবেশ করতে বাধা দেয় এবং যদি মৌলিক কারণগুলি (সংবাদ/ভিশন/হাইপারলিকুইড) আসন্ন পতনের নিশ্চিতি না দেয়, তবে সাধারণ প্রযুক্তিগত কারেকশনে অকালে বেরিয়ে যাওয়া এড়ায়।

### ✨ মূল বৈশিষ্ট্যসমূহ

- **ডুয়াল-টিকার পদ্ধতি**: সূচক বিশ্লেষণ করুন, ETF ট্রেড করুন।
- **T212 লাইভ প্রাইস**: Trading 212 এপিআই (০.২ সেকেন্ড) এর মাধ্যমে রিয়েল-টাইম ইউরো মূল্য পুনরুদ্ধার, সাথে yfinance ফলব্যাক এবং parquet ক্যাশে।
- **Dated Brent স্প্রেড**: ব্রেন্ট স্পট (Dated) এবং ব্রেন্ট ফিউচারসের মধ্যে স্প্রেডের মাধ্যমে ফিজিক্যাল মার্কেটের উত্তেজনা পর্যবেক্ষণ।
- **নেটওয়ার্ক স্থিতিস্থাপকতা**: আলাদা ট্র্যাকার (তথ্য বনাম ডাউনলোড) সহ yfinance সার্কিট ব্রেকার, সমস্ত নেটওয়ার্ক কলেও ১০ সেকেন্ড টাইমআউট।
- **উন্নত কগনিশন**: উন্নত প্রযুক্তিগত/মৌলিক সংশ্লেষণের জন্য **Gemma 4**-এর ব্যবহার।
- **নিউজ এবং ব্লকচেইন সেন্টিমেন্ট**: সামাজিক এবং ফটকামূলক সেন্টিমেন্ট ক্যাপচার করতে **AlphaEar** এবং **Hyperliquid**-এর সংমিশ্রণ।
- **স্বয়ংক্রিয় শিডিউলার**: একটি সার্ভারে নিরবচ্ছিন্নভাবে চালানোর জন্য `schedule.py` স্ক্রিপ্ট (সকাল ৮:৩০ - সন্ধ্যা ৬:০০)।
- **উন্নত ঝুঁকি ব্যবস্থাপনা**: অস্থিরতা এবং বাজারের পরিস্থিতির ওপর ভিত্তি করে স্বয়ংক্রিয় সংকেত সমন্বয়।

### ⚙️ পারফরম্যান্স এবং হার্ডওয়্যার
সিস্টেমটি ডেডিকেটেড জিপিইউ ছাড়াই **সাধারণ হার্ডওয়্যারে পারফরম্যান্স দেওয়ার জন্য** ডিজাইন করা হয়েছে।
- **শুধুমাত্র সিপিইউ**: এলএলএম ইনফারেন্স (ওলামার মাধ্যমে Gemma 4) এবং TimesFM দ্রুত সিপিইউ এক্সিকিউশনের জন্য অপ্টিমাইজ করা হয়েছে যদি পর্যাপ্ত র‍্যাম থাকে।
- **প্রস্তাবিত র‍্যাম**: সর্বনিম্ন ১৬ জিবি (Gemma 4 স্বাচ্ছন্দ্যে চালানোর জন্য ৩২ জিবি পরামর্শ দেওয়া হয়)।
- **এক্সিকিউশন সময়**: একটি পূর্ণ চক্রের জন্য ~২ থেকে ৫ মিনিট (ওয়েব ক্রলিং, এমএল ট্রেনিং, TimesFM প্রেডিকশন এবং ৩টি এলএলএম বিশ্লেষণ সহ)।
- **এপিআই গতি**: আল্ট্রা-ফাস্ট ট্রেডিং ২১২ ইন্টিগ্রেশন (লাইভ প্রাইস পুনরুদ্ধারের জন্য <১ সেকেন্ড)।

## 📂 প্রকল্পের গঠন

উন্নত রক্ষণাবেক্ষণের জন্য প্রকল্পটি মডুলারভাবে সংগঠিত।

```
Trading-AI/
├── src/                     # কোর মডিউল
│   ├── eia_client.py               # জ্বালানি মৌলিক ডেটা ক্লায়েন্ট
│   ├── oil_bench_model.py          # জ্বালানি বিশেষায়িত মডেল
│   ├── enhanced_decision_engine.py # ফিউশন ইঞ্জিন এবং ভিনসেন্ট গ্যানে মডেল
```

## 🚀 দ্রুত শুরু

আপনার স্থানীয় ডেভেলপমেন্ট এনভায়রনমেন্ট সেট আপ করতে এই ধাপগুলি অনুসরণ করুন।

### ✅ পূর্বশর্তসমূহ

- Python 3.12+ (`uv`-এর মাধ্যমে)
- [Ollama](https://ollama.com/) স্থানীয়ভাবে ইনস্টল এবং চালু থাকতে হবে।
- ডাউনলোড করা এলএলএম মডেল: `ollama pull gemma4:e4b`

### ⚙️ ইনস্টলেশন

1.  **রিপোজিটরি ক্লোন করুন:**
    ```sh
    git clone https://github.com/laurentvv/Trading-AI.git
    cd Trading-AI
    ```
2.  **`uv` ইনস্টল করুন (যদি ইতিমধ্যে করা না থাকে):**
    ইনস্টলেশন নির্দেশাবলীর জন্য [astral.sh/uv](https://astral.sh/uv) দেখুন।

3.  **TimesFM ২.৫ ইনস্টল এবং প্যাচ করুন (গুরুত্বপূর্ণ ধাপ):**
    মডেলটিকে `vendor/`-এ ক্লোন করতে এবং প্যাচগুলি প্রয়োগ করতে ইনস্টলেশন স্ক্রিপ্টটি চালান:
    ```bash
    python setup_timesfm.py
    ```

4.  **এনভায়রনমেন্ট শুরু এবং সিনক্রোনাইজ করুন:**
    ```bash
    uv sync
    ```

5.  **ওয়েব রিসার্চের জন্য ব্রাউজার ইনস্টল করুন (Crawl4AI):**
    ```bash
    uv run python -m playwright install chromium
    ```

6.  **আপনার এপিআই কীগুলি কনফিগার করুন:**
    প্রকল্পের রুটে একটি `.env` ফাইল তৈরি করুন:
    ```
    ALPHA_VANTAGE_API_KEY="আপনার_কী"
    EIA_API_KEY="আপনার_কী"
    ```

## 🛠️ ব্যবহার

সিস্টেমটি প্রতিটি এক্সিকিউশনের আগে সাম্প্রতিক ডেটার ওপর তার মডেলগুলিকে প্রশিক্ষণ দেয় এবং তারপর একটি সিদ্ধান্ত দেয়।

### সিমুলেশন মোড (পেপার ট্রেডিং)

১০০০ ইউরোর কাল্পনিক মূলধন দিয়ে কোনো ঝুঁকি ছাড়াই সিস্টেমটি পরীক্ষা করতে `--simul` ফ্ল্যাগ ব্যবহার করুন। সিস্টেমটি কেনা এবং বিক্রির একটি কঠোর ইতিহাস পরিচালনা করবে।

```sh
# একটি সিমুলেটেড বিশ্লেষণ চালান (ডিফল্ট: SXRV.DE - Nasdaq 100 EUR)
uv run main.py --simul

# তেলের ওপর চালান (WTI)
uv run main.py --ticker CRUDP.PA --simul
```

### বাস্তব সম্পাদন (Trading 212)

সিস্টেমটি এখন Trading 212-এর সাথে **সম্পূর্ণভাবে একীভূত**:
- **পোর্টফোলিও যাচাইকরণ**: কোনো পদক্ষেপ নেওয়ার আগে রোবটটি আপনার প্রকৃত ক্যাশ এবং পজিশনগুলি পরীক্ষা করে।
- **এপিআই ম্যানেজমেন্ট**: অনুরোধের সীমার (Rate Limiting) বিরুদ্ধে স্বয়ংক্রিয় পুনরায় চেষ্টা করার ব্যবস্থা অন্তর্ভুক্ত রয়েছে।

```sh
# বাস্তব সম্পাদন সহ বিশ্লেষণ চালান (.env অনুযায়ী ডেমো বা রিয়েল)
uv run main.py --t212
```
